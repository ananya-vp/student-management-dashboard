# Student Management Dashboard

A React-based Student Management Dashboard that allows:

- Add Students
- Edit Students
- Delete Students
- Manage Advance and Bonus Components
- Persistent Data using Local Storage

## Tech Stack
- React.js
- JavaScript
- CSS

## Features
- Dynamic Student Management
- Data stored in Local Storage
- Clean UI
- Fully Functional CRUD Operations

## How to Run the Project

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
